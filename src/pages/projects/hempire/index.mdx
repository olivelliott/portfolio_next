## Languages 

ReactJS, NodeJS, NextJS, TailwindCSS

## Description

A responsive single page portfolio created for a local band to showcase their music and have
a portfolio to present to booking agents and recording studios. It is my first post graduate project 
that exemplifies my proactive desire to develop diverse skills. Self taught use of NextJS and TailwindCSS 
that highlights eagerness to excel in learning new languages.


## User Story
```
AS A small town band,
I WANT to showcase my music, shows, and merchandise
SO THAT I can have a portfolio to show booking agents,
as well as increase attendance at my shows and sell my merchandise.
```
